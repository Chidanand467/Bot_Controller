AUTONOMOUS PERSON FOLLOWING ROBOT

AI-Powered Real-time Tracking System with Object-Oriented Architecture
KEY INNOVATION: PID-Controlled Navigation Algorithm

An intelligent robotic system that combines computer vision, embedded systems, and object-oriented programming to track and follow humans in real-time. Using Raspberry Pi as the central brain and smartphone camera for vision input, the system processes live video streams to detect and follow a target person autonomously.

SYSTEM ARCHITECTURE

Raspberry Pi 4

Quad-core 64-bit processor with 4GB RAM running Python with OpenCV for real-time processing

Vision System

Smartphone IP Camera streaming at 30 FPS with WiFi connectivity for high-quality video input

Motor Control

Dual 12V DC Motors with ESCs for precise speed and direction control

AI Processing

Haar Cascade & Deep Learning models for real-time person detection and tracking

OOP IMPLEMENTATION

Classes & Objects

Robot, Camera, MotorController, FaceDetector, and Navigator classes with proper instantiation

Encapsulation

Private attributes with getter/setter methods for hardware control and safety protocols

Abstraction

Complex computer vision algorithms hidden behind simple interface methods

Inheritance

Base Robot class extended to PersonFollowerRobot with specialized tracking methods

Technical Stack

Python 3.9 OpenCV 4.5 NumPy RPi.GPIO Socket Programming Multithreading

WORKING ALGORITHM

1

Frame Capture

30 FPS video streaming via IP camera with WiFi connectivity for continuous input

2

Face Detection

Haar Cascade classifiers identify human faces in the frame with high accuracy

3

PID Control

Proportional-Integral-Derivative algorithm calculates optimal motor speeds for smooth following

4

Motor Control

PWM signals control ESC modules for precise movement and direction changes
